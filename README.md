# postgrest-url 
> Build urls for PostgREST

This is for advanced users.


## Install

```sh
$ npm install --save postgrest-url
```

## Usage
Check the examples and code in the test folder.

## Build and test
```sh

npm run watch -- -s

npm test -s
```

## Release and publish
https://github.com/conventional-changelog/standard-version

```sh
npm run release
npm run publish
```
